## 📦 Избранные проекты

* 🏰 **WardenCraft (Закрытый код)** — Масштабная тотальная конверсия движка Minecraft 1.7.10 в **полноценную Open-World MMORPG**. Полностью вырезана песочница, создан открытый мир с MMORPG-механиками. 
  * Выступаю в роли Lead-разработчика. С нуля спроектировал и реализовал экосистему: Web-сайт (EJS) ↔ API ↔ Игровой сервер (Java) ↔ Клиент игры.
  * Глубоко переписал графический движок, систему рендера чанков, памяти и многопоточности.
  * Разработал кроссплатформенный лаунчер на **Electron** с автообновлением клиенты игры и проверкой хэшей файлов.

* 🛡️ **[Golem Gateway](https://github.com/GrishaDeLumiere/golem-gateway)** — Самописный Node.js шлюз для LLM. Перехватывает веб-сессии через Puppeteer/OAuth2 и оборачивает их в стандарт OpenAI API. Включает локальную веб-панель на Vanilla JS с динамическим выгрузкой/загрузкой провайдеров из ОЗУ и реал-тайм апдейтером ядра.

* 🌻 **[PVZ PLUS](https://github.com/GrishaDeLumiere/PVZGame)** — Клон механик Plants vs. Zombies на Java. Разработан кастомный 2D-рендер с поддержкой GLSL-шейдеров, пространственный звук на OpenAL и собственная система GUI *(Проект заморожен)*.
